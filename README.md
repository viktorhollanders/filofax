# Address Book assignment for web-programming-II

An address book that can store both company addresses and personal addresses. Runs on Vite with typescript and uses UI Kit for styling

## Author
Viktor Hollanders
[viktorh23@ru.is](mailto:viktorh23@ru.is)

## Prerequisites
- Node.js v18+
- pnpm v8+

## Getting started 
Install using pnpm 
```bash
pnpm install 
```

Then, run the development server:
```bash
pnpm run dev 
```

Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.

## UIKit
This project uses UI Kit so that students can get experience using it.
[See here for more info](https://getuikit.com/docs/installation)
